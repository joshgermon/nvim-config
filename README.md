# Neovim Config

## Explorer

- `<leader>e` - Explorer (mini.files)

## Window Navigation

- `<C-h>` - Move to left window
- `<C-j>` - Move to lower window
- `<C-k>` - Move to upper window
- `<C-l>` - Move to right window

## Search

- `<Esc>` - Escape and clear hlsearch

## Picker (mini.pick)

- `<leader>ff` - Find Files
- `<leader>sg` - Grep
- `<leader>sh` - Help Pages
- `<leader>sb` - Buffers
- `<leader>so` - Recent Files
- `<leader>sc` - Colorschemes
- `<leader>sd` - Diagnostics
- `<leader>sw` - Word under cursor

## Git

- `<leader>gd` - Git Diffview
- `<leader>gh` - File History

## Toggle

- `<leader>th` - Toggle Inlay Hints
- `<leader>td` - Toggle Diagnostics

## LSP / Mason

- `<leader>cm` - Mason
- `<leader>cf` - Format
- `<leader>cl` - Lint
